h1. What is it ? 

This project is an implementation of the "JSR 330":http://code.google.com/p/atinject/ specification.

Actually there is no version deployed in a Maven respository. To use this project you have to download and build it from its sources with Maven. Run <code>mvn clean install</code> from the root directory. To use this project with Maven or Ivy, add the following lines with the correct version.

* API

<pre>
<code><dependency>
   <groupId>com.my.container</groupId>
   <artifactId>container-api</artifactId>
   <version>project-version</version>
</dependency></code>
</pre>

* Core

<pre>
<code><dependency>
   <groupId>com.my.container</groupId>
   <artifactId>container-core</artifactId>
   <version>project-version</version>
</dependency></code>
</pre>

*Note :* if you want an example, you can have a look at the container-sample project.	

h2. System Requirements

JDK 1.5 or above.
